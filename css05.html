구조선택자
:first-child - 여러 태그(형제) 중에서 첫번째에 위치하는 태그
:last-child - 형제 중에서 마지막에 위치하는 태그
:nth-child(n) - 형제중 앞에서부터 n번째에 해당하는 태그
:nth-last-child(n) - 형제중 뒤에서부터 n번째에 해당하는 태그

형태 구조 선택자
:first-of-type - 형제 관계 중에서 첫 번째로 등장하는 특정 태그
:last-of-type - 형제 관계 중에서 마지막으로 등장하는 특정 태그
:nth-of-type(n) - 형제 관계 중에서 n번째로 등장하는 특정 태그
:nth-last-of-type(n) - 형제 관계 중에서 뒤에서부터 n번째로 등장하는 특정 태그

차이점 구조선택자 : 존속 개념이 있음 (span, li) 형태 구조 선택자 : 존속 개념이 없음 (h, p)
